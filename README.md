# KHOKAI ERP Lite / Partner POS V2.1.4

อัปเดตหน้าเปิดออเดอร์ด่วนให้ใช้งานบนมือถือเร็วขึ้น:

- เอาช่องค้นหา / SKU / 옵션ID ออกจากหน้าแรก
- เลือกสินค้าจากรายการตรง ๆ เพราะสินค้ามีไม่เยอะ
- หน่วยสินค้าเปลี่ยนเป็นปุ่ม: แพ็ก/ชิ้น หรือ ลัง
- จำนวนเลือกเร็ว 1, 2, 3, 5, 10
- มีปุ่ม + / - ข้างจำนวน
- ยังรองรับจำนวนต่อลังแบบพัสดุและรถส่งถึงที่
