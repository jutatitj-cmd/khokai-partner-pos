# KHOKAI Partner POS V17.0.8.2

ฐานจาก V17.0.8 ที่ใช้งานได้จริง

อัปเดต:
- ตั้งค่าเริ่มต้นช่องทางจัดส่งเป็น 택배
- ปรับข้อความส่งลูกค้าตามฟอร์แมตใหม่
- ค่าส่งเริ่มต้น 5,000 วอน

## V17.0.8.2

- ปุ่มคัดลอกในหน้าแก้ไขแยกไทย/เกาหลี
- ข้อความเลือกตามสถานะออเดอร์: TEMP = ขอชำระเงิน, PAID = ข้อความหลังชำระเงิน
